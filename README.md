# Project 19, Quote App CLI 🧑‍💻. (April 19th - 20th)

- A simple command-line app that fetches random quotes using an API and lets you build a temporary quote collection.

## 🧠 About

- This project was built as part of my Python learning journey, focusing on working with APIs and building interactive CLI programs.

- It started as a basic script to fetch quotes, and I expanded it with a menu system and a collection feature to make it more interactive.

## 🚀 Features

- Generate random quotes from an API
- Display quote + author in a clean format
- Save quotes to a temporary collection (runtime only)
- View collected quotes
- Simple CLI menu system

## 🛠️ Tech Used

- Python
- requests (API calls)

## 📚 What I Learned

- How to fetch and handle API responses
- Working with JSON data returned from APIs
- Structuring a CLI application
- Managing user input and program flow

## ⚠️ Notes

Quotes are not saved permanently (they reset when the program restarts).

## 🔮 Future Improvements

- Save quotes to a JSON file
- Add delete/clear collection option
- Improve error handling
